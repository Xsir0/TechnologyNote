### 脚本的概述

- 脚本是指以解释执行方式的程序语句集合体

### 脚本的控制结构

- 程序的控制结构使得程序语句的执行顺序能够有条件地智能改变
- 控制结构有顺序结构、分支结构和循环结构

### 分支结构

- 分支结构根据条件判断的真假来选择执行某一分支的语句体
- 分支结构的实现方式：
  - `&&、||`符号级联
  - `if-else`语句

```
 if [ 条件判断 ]
 then
     真分支结构
 elif [ 条件语句 ]
 then

 else
     假分支结构
fi
```

```
- `case`语句
```

```
case 分支
    a) echo "test1"
    ;;
    *) echo "default "
    ;;
esac
    
```

### 循环结构

- 循环结构根据条件判断的真假来确定是否执行循环体内的语句
- 循环结构实现方式：
  - `while`语句

```
while [ 条件 ]
do
    echo "while test "

done
```
